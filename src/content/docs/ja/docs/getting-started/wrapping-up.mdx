クイックスタートガイドを完了したことで、Capgoの主要な概念について基本的な理解が得られたはずです！このガイドで学んだ主要な概念は以下の通りです：

<Steps>
1. CodePushGoアカウントへのアプリの追加

2. CIパイプラインへのCapgoの統合

3. 新しいコミットでCapgoへのバンドルアップロードをトリガーする

4. CodePushGoSDKでライブアップデートを有効にするためのアプリの設定

5. CodePushGoダッシュボードからアプリへのライブアップデートのデプロイ
</Steps>
