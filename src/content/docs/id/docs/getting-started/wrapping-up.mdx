Sekarang setelah Anda menyelesaikan panduan quickstart, Anda seharusnya memiliki pemahaman dasar tentang konsep-konsep kunci CodePushGo! Konsep-konsep kunci yang telah Anda pelajari dalam panduan ini adalah:

<Steps>
1. Menambahkan aplikasi ke akun CodePushGo Anda

2. Mengintegrasikan CodePushGo dengan pipeline CI/CD Anda

3. Memicu unggahan bundle ke CodePushGo pada commit baru

4. Mengkonfigurasi aplikasi Anda untuk mengaktifkan pembaruan langsung dengan SDK CodePushGo

5. Menerapkan pembaruan langsung ke aplikasi Anda dari dashboard CodePushGo
</Steps>
