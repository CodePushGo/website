`npx @capgo/cli@latest init [apikey]`

Este comando te guía paso a paso para:

- Agregar tu aplicación a CodePushGo
- Agregar el código a tu aplicación para validar la actualización
- Construir tu aplicación
- Subir tu aplicación a CodePushGo
- Ayudarte a verificar si la actualización funciona
