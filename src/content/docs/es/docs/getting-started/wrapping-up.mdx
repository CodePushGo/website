Ahora que has completado la guía de inicio rápido, deberías tener una comprensión básica de los conceptos clave de CodePushGo. Los conceptos clave que has aprendido en esta guía son:

<Steps>
1. Agregar una aplicación a tu cuenta de CodePushGo

2. Integrar CodePushGo con tu pipeline de CI/CD

3. Activar las subidas de paquetes a CodePushGo con nuevos commits

4. Configurar tu aplicación para habilitar actualizaciones en vivo con el SDK de CodePushGo

5. Desplegar actualizaciones en vivo a tu aplicación desde el panel de control de CodePushGo
</Steps>
