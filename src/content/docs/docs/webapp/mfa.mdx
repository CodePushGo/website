Two-factor authentication protects your CodePushGo console account before someone can create API keys, upload bundles, or change channels.

## Requirements

- A phone or hardware authenticator that supports one-time codes.
- Access to [console.codepushgo.com](https://console.codepushgo.com).

## Enable 2FA

1. Open [console.codepushgo.com/dashboard/settings/account](https://console.codepushgo.com/dashboard/settings/account).
2. Choose the two-factor authentication action.
3. Scan the QR code with your authenticator app.
4. Enter the current code to verify setup.

Do not share the QR code or recovery codes. Anyone with them can access the account.

## Login with 2FA

After password login, enter the current authenticator code before it expires. If the code changes while you are typing, use the new one.

## Disable 2FA

Open account settings and choose the disable action. Disable 2FA only when you are replacing the authenticator or recovering account access.
