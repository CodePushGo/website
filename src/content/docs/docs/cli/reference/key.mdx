The `key` command lets you manage your app signing keys.

### save

`npx @codepushgo/cli key save`

Saves a base64-encoded encryption key to the React Native config (useful for CI).

Options:

- `-f, --force`: Force generation of a new key
- `--key`: Path to the key file to save in the React Native config
- `--key-data`: Key data to save directly in the React Native config

### create

`npx @codepushgo/cli key create`

Creates a new encryption key.

Options:

- `-f, --force`: Force generation of a new key

### delete_old

`npx @codepushgo/cli key delete_old`

Deletes the old encryption key.
